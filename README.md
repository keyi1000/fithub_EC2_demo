# Fithub バックエンド

このリポジトリは、**Fithub プロジェクトのバックエンドサービス**です。以下の技術スタックを使用しています：

- **Express v5**
- **TypeScript**
- **Node.js v20.18.1**
- **npm v11.1.0**

## 開発メモ

コミット前に必ず以下のコマンドで**コード品質のチェック**を行ってください：

```bash
# Lint チェック
npm run lint

# Prettier フォーマット チェック
npm run prettier

# 両方まとめて実行
npm run check
```

他のスクリプトやコマンドについては、**package.json** の **scripts** セクションをご確認ください。
